## 基本概念


### working-tree

local だけで

- working-tree     : 作業ファイル ( 開発するときに修正するファイル )
- index / staging  : add    で追加される
- local repository : commit で追加される

の 3つがある



### HEAD とは

`HEAD` はその repository/branch の最新の commit のこと
