
# git branch


## branch list を表示

```
git branch
```


## コミットの先端情報も表示したい場合

```
git branch -v
```


## branch を作る

```
git branch branch_name
```


## branch name を変更する

```
git branch -m name_old name_new
```

```
git branch -m master main
```


## branch を切り替える ( checkout ? )

```
git checkout branch_name
```
