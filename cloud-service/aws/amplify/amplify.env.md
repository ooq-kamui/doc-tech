
# amplify env


## env の list 表示

```
amplify env list
```


## env の 追加

```
amplify env add dev
```

## env の 切り替え

```
amplify env checkout dev
```


## env の 詳細情報 表示

```
amplify env get
```


## cloud 上にある env を pull

```
amplify env pull
```
