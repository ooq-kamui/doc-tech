
#  fd

## arg

```
fd ptn dir
```

## option

```
-e : 拡張子指定
     複数の場合は -e も複数指定

-t : ファイルタイプ指定

-E : 除外パターン

-I : .gitignore, .ignore, .fdignore を含める
```


```
sort option : 不明 ( ない? )
```


## ptn
正規表現が使える

複数指定
```
fd 'aaa|bbb'
```
